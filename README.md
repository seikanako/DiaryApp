## DiaryApp
#### 開発環境
- Xcode 7.1.1
- Deployment Target iOS 8.0
- Swift 2.0
- 実機確認 iPhone6

#### 説明
@nifty mobile backendを使用したiOSアプリです。
Navigation Controllerを使用し、1ページ目は登録済の日記の一覧、２ページ目は日記の登録画面になっています。

以下のサンプルアプリを参考に作りました。
